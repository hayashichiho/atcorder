1. 問題のディレクトリを作成し、A問題ディレクトリに移動する
-------------------------------------------------
acc new abc300
cd abc300/a

2. コードを書く
-------------------------------------------------
code main.py

3. サンプルテストを自動で実行する
-------------------------------------------------
oj test -c "python3 main.py"

4. 提出する（ブラウザで行う）
-------------------------------------------------
# AtCoderの提出ページを開き、main.pyの内容をコピペして提出

5. 次の問題（B問題）を解く場合
-------------------------------------------------
cd ../b && code main.py


6. 単発の問題（例：Tessoku BookのAI問題）をダウンロードする場合
-------------------------------------------------
oj d https://atcoder.jp/contests/tessoku-book/tasks/math_and_algorithm_ai
code main.py
